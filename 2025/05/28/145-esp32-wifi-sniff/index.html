Wi-Fi 协议模式

目前，IDF 支持以下协议模式：

协议模式 描述
802.11b 调用函数 esp_wifi_set_protocol(ifx, WIFI_PROTOCOL_11B)，将 station/AP 设置为仅 802.11b 模式。
802.11bg 调用函数 `esp_wifi_set_protocol(ifx, WIFI_PROTOCOL_11B
802.11g 调用函数 `esp_wifi_set_protocol(ifx, WIFI_PROTOCOL_11B
802.11bgn 调用函数 `esp_wifi_set_protocol(ifx, WIFI_PROTOCOL_11B
802.11gn 调用函数 `esp_wifi_set_protocol(ifx, WIFI_PROTOCOL_11B
802.11 BGNLR 调用函数 `esp_wifi_set_protocol(ifx, WIFI_PROTOCOL_11B
802.11 LR 调用函数 esp_wifi_set_protocol(ifx, WIFI_PROTOCOL_LR)，将 station/AP 设置为 LR 模式。此模式是乐鑫的专利模式，可以达到 1 公里视线范围。请确保 station 和 AP 同时连接至 ESP 设备。

Wi-Fi Sniffer 模式

Wi-Fi Sniffer 模式可以通过 esp_wifi_set_promiscuous() 使能。如果使能 Sniffer 模式， 可以 向应用程序转储以下数据包。

  • 802.11 管理帧
  • 802.11 数据帧，包括 MPDU、AMPDU、AMSDU 等
  • 802.11 MIMO 帧，Sniffer 模式仅转储 MIMO 帧的长度。
  • 802.11 控制帧
  • 802.11 CRC 错误帧

不可以 向应用程序转储以下数据包。

  • 802.11 其它错误帧

void wifi_promiscuous_cb(void *buf, wifi_promiscuous_pkt_type_t type) {
    if (type != WIFI_PKT_MGMT) return; // 只处理管理帧
    
    const wifi_promiscuous_pkt_t *promiscuous_pkt = (wifi_promiscuous_pkt_t *)buf;
    const wifi_ieee80211_packet_t *ipkt = (wifi_ieee80211_packet_t *)promiscuous_pkt->payload;
    const wifi_ieee80211_mac_hdr_t *hdr = &ipkt->hdr;
    
    // Beacon帧类型为0x80
    if (hdr->frame_ctrl[0] == 0x80) {
        // 这里处理Beacon数据
        // - SSID
        // - 信号强度（RSSI）
        // - 信道信息
        // - 其他IE（Information Elements）
    }
}

// 2. 初始化WiFi并设置为混杂模式
void init_wifi_promiscuous(void) {
    wifi_init_config_t cfg = WIFI_INIT_CONFIG_DEFAULT();
    ESP_ERROR_CHECK(esp_wifi_init(&cfg));
    ESP_ERROR_CHECK(esp_wifi_set_mode(WIFI_MODE_STA));
    ESP_ERROR_CHECK(esp_wifi_start());
    
    // 注册回调函数
    esp_wifi_set_promiscuous_rx_cb(wifi_promiscuous_cb);
    // 启用混杂模式
    esp_wifi_set_promiscuous(true);
    // 设置信道（可选，也可以进行信道扫描）
    esp_wifi_set_channel(1, WIFI_SECOND_CHAN_NONE);
}
