典型电路

测量方法

  1. 在已知条件下标定R0，例如，结晶空气或者200ppm的乙醇气体里，预热后读取电压计算Rs为R0

    ​ Rs=Rl*(Vc-Vout)/Vout

    本次是R0=20*(3300-944)/944=49.92K

  2. 每次读取计算Rs，得到Rs/R0，本次是0.4

  3. 代入曲线读取浓度

曲线是如何拟合的

1. 数据来源

从数据手册的“灵敏度曲线”读出若干数据点：

  • 横轴：气体浓度 CCC（ppm，log 坐标）
  • 纵轴：比值 Rs/R0Rs/R0Rs/R0（log 坐标）

得到一组实验点 (C,Rs/R0)(C, Rs/R0)(C,Rs/R0)。

2. 坐标变换

将数据点转换到对数坐标系：

x=log⁡10(C), y=log⁡10(Rs/R0)

这样原本的曲线在 log-log 坐标下大致呈直线。

3. 线性拟合

在 log-log 坐标下，曲线近似为一条直线

**y=m⋅x+c **

使用最小二乘法（例如 numpy.polyfit）求解斜率 m 和截距 c。

4. 转换回原公式

5. 总结

  • 通过 log-log 坐标变换，可以把非线性关系近似为直线；
  • 用线性拟合得到的斜率和截距，再还原成幂律公式；

适用于一定浓度范围内的酒精浓度估算。

PPM

  • ppm 是 parts per million 的缩写，意思是“百万分之一”。

  • 它是一种浓度的相对单位，表示某种物质在混合物中的比例。

Demo

#include <math.h>

// 硬件参数
static const float Vc = 3.3f;
static const float RL = 20000.0f;

// 用这次空气读数 909 mV 计算得到的 R0（建议存储在设备的非易失存储里）
static const float R0 = 52600.0f; // ≈52.6 kΩ

// 拟合得到的系数（1–100 ppm 适用，基于手册曲线目读点）
static const float a = -2.1869f;
static const float k = 1.1174f;

float ethanol_ppm_from_Vout(float Vout) {
    if (Vout <= 0.0f) return 0.0f;
    float Rs = RL * (Vc - Vout) / Vout;
    float ratio = Rs / R0;
    if (ratio <= 0.0f) return 0.0f;
    return k * powf(ratio, a);
}

总结

实际使用需要提前测出当前洁净空气下的R0，然后再进行测试
