无法使用主机代理

1.公共网络的防火墙需要关闭

检查 v2ray 是否监听 0.0.0.0（支持局域网访问）

打开 v2rayN → 设置 → 代理端口 → 确保监听地址为：

0.0.0.0:10808

不要是 127.0.0.1:10808（仅限本机）

你也可以在主机上执行（CMD）：

netstat -an | findstr 10808

应该能看到：

TCP    0.0.0.0:10808       LISTENING

添加规则放行 VMware 和 ping

如果你不希望关闭防火墙，可以用管理员 PowerShell 添加规则：

# 放行 v2ray 端口
New-NetFirewallRule -DisplayName "Allow v2ray 10808" -Direction Inbound -Protocol TCP -LocalPort 10808 -Action Allow -InterfaceAlias "VMware Network Adapter VMnet8"

# 放行 ping（ICMPv4）
New-NetFirewallRule -DisplayName "Allow ICMPv4 ping from VM" -Protocol ICMPv4 -IcmpType 8 -Direction Inbound -Action Allow -InterfaceAlias "VMware Network Adapter VMnet8"

运行完之后，重新测试 ping 和 curl。
