openclaw玩一哈

openclaw

最后还是要搞来玩一下，一开始以为配置很复杂，然后用了大佬的一键脚本bash <(curl -sL kejilion.sh) app openclaw，但是夹带的东西太多了，有点烦，于是去找教程了

环境

  • 波兰Nat服务器
  • Debian12
  • TG Bot

配置

申请TG Bot

1.向BotFather发送/newbot，建好bot后发送/mybots查看token

安装openclaw

  • 官方文档

  • 官方快速安装脚本curl -fsSL https://openclaw.ai/install.sh | bash

运行新手引导向导（并安装服务）

openclaw onboard --install-daemon

根据指引填入刚刚获取Bot的token，后面向bot发送/start获取Pairing code填入

或者完成安装后手动添加

openclaw pairing approve telegram "paring code"

启动 Gateway 网关

openclaw gateway status

快速验证

openclaw status
openclaw health
openclaw security audit --deep
